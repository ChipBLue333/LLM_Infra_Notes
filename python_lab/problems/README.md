# Problems by Topic

题目现在按训练专题真实分类存放。

## 目录

- `01_array_hash_map_basics/`: 数组、哈希表、基础状态维护
- `02_stack/`: 栈
- `03_two_pointers_sliding_window_binary_search/`: 双指针、滑动窗口、二分
- `04_linked_list/`: 链表
- `05_binary_tree/`: 二叉树
- `06_backtracking/`: 回溯
- `07_graph_bfs_dfs/`: 图、BFS、DFS
- `08_dynamic_programming_basic_grid/`: 基础 DP、网格 DP
- `09_dynamic_programming_knapsack_string/`: 背包 DP、字符串 DP
- `10_greedy_intervals/`: 贪心、区间
- `11_heap/`: 堆、Top K、K-way merge、双堆
- `12_union_find/`: 并查集
- `13_topological_sort/`: 拓扑排序、课程依赖
- `14_trie/`: Trie、前缀树、字符串前缀查询

## 复习方式

1. 进入对应专题目录。
2. 按题号顺序重做。
3. 先独立写实现，再运行单题测试。
4. 失败后先自己定位，再请求 review。
